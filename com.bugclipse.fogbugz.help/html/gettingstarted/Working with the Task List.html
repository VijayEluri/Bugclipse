Working with the Task List


  • The new query created is listed in the task list view and the cases that belong to the selected filter are displayed as tasks under the query
  • The tasks are sorted in the order of decreasing priority, indicated by the overlay icon to the left of each task. FogBugz priority indications have been mapped as Very High(1),High(2),Normal(3,4),Low(5,6),Very Low(7)
  • Tasks that have been set with a due date have a clock icon on the top-left.
  • The task category (Bug/Feature/Inquiry/Schedule Item) is indicated by the overlay icon at the center of each task
  • The F5 key synchronizes the query/task with the repository. The scheduled synchronization brings up new tasks into the task list automatically. The frequency of the automatic synchronization can be set in the preferences dialog Window -> Preferences -> Mylyn -> Task List
  • The New Task button brings up a rich editor for entering new task information into the FogBugz repository
